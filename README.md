## Setup & Deployment Instructions

- Clone the repo
- Run `npm install`
- Make modifications in `app` folder files
- Keep you images assets in `public > images`
- Follow these [instructions](https://nextjs.org/learn/basics/deploying-nextjs-app/deploy)
